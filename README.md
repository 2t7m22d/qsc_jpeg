3、在过程中，你遇到最大的困难是什么？你是怎么解决的？
    我遇到的最大的困难是没有霍夫曼编码表，一开始没有仔细看文档，打算把整个霍夫曼编码表写进程序，后来发觉整个写进去时间也太长了，而且后面的表都不好写入，
然后就读了几遍范式霍夫曼表那个，意识到能有一张更简约的表（一开始我以为范式霍夫曼表和后面给的那个表不一样，查JPEG霍夫曼表的时候发现是一样的qaq），然后
一开始打算自己写把那个霍夫曼表写出来，然后发现给的那个没有按顺序排，然后用excel统计的话又不知道怎么导进去。然后就继续后面文件头的文档，发现文件头
里有说包含了霍夫曼表，然后就新建了一个jpeg文件在文件头里找，然后在网上找有什么标记，就FF C4后面部分就是霍夫曼表然后就找到了，然后就方便很多。虽然但是
在过程中还有别的好多困难qaq，但是这个感觉简单又解决不了的东西就很让人生气。
